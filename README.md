
# Roxy – Voice-based Conversational AI Agent (Prototype)

Roxy is a real-time, voice-powered AI assistant built using Vapi.ai and a Python backend (FastAPI), deployed on Replit. The project explores personality-driven human–AI interaction.

## Tech Stack
- Vapi.ai
- Python
- FastAPI
- Replit

## Features
- Real-time voice interaction
- Personality-based responses (sarcastic, motivational)
- Custom prompt design

## Status
Prototype project for learning and experimentation.

NOTE: the live demo is currently inactive due to expired vapi free credits.
